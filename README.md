# Gaia Plastics Project
Repository for Gaia Plastics, an IoT project for counting plastic objects in rivers using computer vision.

This README provides a brief project description and setup information. Please go to our Wiki (https://github.com/waterhog/gaia_plastics/wiki) if you want more in-depth information about the project and how to set up a device.
